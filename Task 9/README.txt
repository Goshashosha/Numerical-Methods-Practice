Данная программа предназначена для приближенного решения краевых задач вида:
u''(t) = f(t, u), 0 < x < l  (1)
u(0) = mu1, u(l) = mu2

Реализован метод Ньютона для итерационного решения нелинейной сеточной задачи.
Краткий обзор метода Ньютона:
1) Строится разностная схема для уравнения  (1)
-- Правая часть уравнения приближается следующим образом:
-- f(u_n[i + 1]) ~ f(u_n[i]) + f'(u_n[i]) (u_n[i + 1] - u_n[i]),
-- где n - индекс точки в разбиении, i - номер итерации.
2) Решается система линейных алгебраических уравнений, результатом является
-- i-ое приближение.
3) Проводится оценка сходимости по норме при заданном параметре:
-- ||u[i] - u[i-1]|| < eps
-- Если условие выполнено - решение получено. В противном случае, процесс продолжается.
В данном случае решение построено на равномерной сетке.

В качестве примера работы программы решается следующая задача:
u''(t) = -e ^ u(t), 0 < x < 1
u(0) = 0, u(1) = 0

Также проведено исследование сходимости итерационного метода на данном примере при изменении
погрешности.
